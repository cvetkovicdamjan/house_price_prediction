# House Price Prediction in PyTorch
![House Price Prediction Image](image/house_price_prediction.jpg)
PyTorch model to predict house prices based only on features such as square footage, number of bedrooms, and other features.

## Data
Data is available from the Kaggle competition: [House Prices - Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques)
